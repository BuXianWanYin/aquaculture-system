## 快速导航

### 主要功能模块
1. **用户与权限管理模块** - 用户认证、角色权限控制
2. **基础数据管理模块** - 品种、区域、设备管理
3. **养殖计划管理模块** - 计划录入、审批、调整
4. **产量统计管理模块** - 产量录入、审核、完成率分析
5. **数据报表与分析模块** - 多维度数据统计与可视化
6. **操作日志与追溯模块** - 操作记录与追溯

### 次要功能模块
1. **消息通知模块** - 站内通知、消息管理
